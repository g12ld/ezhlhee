# AGENTS.md — إزهلها (Ezhalha)

## Project Overview
 رابط الموقع : https://www.ezhalhe-sa.com/
إزهلها هي وكالة سعودية متخصصة في تصميم وإنشاء وتطوير المتاجر الإلكترونية الاحترافية، مع تركيز كامل على منصتي **سلة** و **زد**، بالإضافة إلى WooCommerce.

الموقع يخدم السوق السعودي فقط تقريباً، ويستهدف أصحاب المشاريع الذين يريدون بدء التجارة الإلكترونية أو تطوير متاجرهم الحالية.
